## Audit 检查项（与脚本一致）

1. frontmatter 包含 `name` 与 `description`，且 `name` 与目录名一致。
2. SKILL.md 包含必需 H2：
   - `## When to Use`
   - `## Workflow`
   - `## Verification`
   - `## Common Rationalizations`（v0.2.0 起，ADR-002 D9）
3. SKILL.md **不**包含禁止 H2：`## 和其他 Skill 的区别`（v0.2.0 起，ADR-002 D9）。
4. SKILL.md 主文件 < 500 行（advisory，仅 warn）。
