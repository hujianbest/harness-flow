1. **FR-008 内容质量评分**：补充说明

### 建议修订（Minor）

2. **补充性能指标的网络条件**：
   ```
   - 首屏加载时间：4G ≤ 2s，5G ≤ 1s，WiFi ≤ 1s
   - 文章加载时间：4G ≤ 1s，5G ≤ 0.5s，WiFi ≤ 0.5s
   ```

3. **补充安全限流数值**：
   ```
   - 用户注册：每 IP 每小时 5 次
   - API 调用：每用户每分钟 100 次
   - 搜索请求：每用户每分钟 10 次
   ```

4. **补充移动端 UI 测试**：
   ```
   - iOS: XCUITest
   - Android: Espresso
   - 跨平台: Appium
   ```

5. **明确开放问题**（建议）：
   - 技术选型：推荐 Flutter（跨平台）、OpenAI API（MVP 阶段）
   - 推荐系统：MVP 用规则（用户兴趣标签 + 时效性），v2.0 引入协同过滤
   - 内容来源：初期使用公开 RSS，无需授权
